# eRopCssGlobe Release Notes

## v1.0.0 (2026-04-10)
- Initial release
- CSS-only rotating globe (no JavaScript, no Canvas, no SVG)
- Stylized continent shapes using `border-radius` div elements
- Scrolling map track with seamless loop (`translateX` animation)
- Atmospheric glow, 3-D sphere shading, arctic/antarctic ice caps
- Latitude grid lines and meridian grid lines (CSS `repeating-linear-gradient`)
- Configurable rotation speed: Fast (12s) / Normal (28s) / Slow (50s)
- Star field background
- Dark ocean depth gradient
