# eRopShinhanBanner Release Notes

## v1.0.0 (2026-03-30)

### 신규 기능
- 신한금융그룹 공식 로고를 활용한 가로형 브랜딩 배너 위젯
- 브랜드 컬러 (#0046BE 로열 블루) 기반 다크 테마 디자인
- CSS 애니메이션 효과:
  - 배경 그라디언트 플로우 (8s 사이클)
  - 로고 글로우 펄스 (3s 사이클)
  - 로고 플로팅 (4s 사이클)
  - 파티클 상승 효과 (3~7s 랜덤)
  - 코너 장식 펄스 (3s, 각 0.5s 오프셋)
  - 수평 라인 시머 (3s 사이클)
  - 하단 스크롤 티커 (32s 사이클)
  - 배지 팝인 애니메이션
  - 슬라이드인 텍스트 애니메이션
  - SOC 상태 표시등 블링크 (2s 사이클)
  - 스캔라인 오버레이
  - 좌우 광선 이펙트
- 설정 가능 항목: 위젯 제목, 배너 메인 타이틀, 스크롤 티커 텍스트
- 다국어 지원: 한국어, 영어, 일본어, 중국어(간체)
