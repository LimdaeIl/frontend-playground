# eRopKeimyungBanner Release Notes

## v1.0.0 (2026-03-30)

### 신규 기능
- 계명대학교 공식 로고를 활용한 가로형 브랜딩 배너 위젯
- 브랜드 컬러 기반 디자인
  - 딥 네이비: #1B3A7C (주조색)
  - 스칼렛 레드: #C8102E (보조색)
  - 골드: #D4A843 (포인트색)
- CSS 애니메이션 효과:
  - 배경 딥 네이비 그라디언트 플로우 (9s 사이클)
  - 상/하단 레드 스트라이프 펄스
  - 로고 3색 글로우 사이클 (네이비→레드→골드, 5s)
  - 로고 플로팅 (4s)
  - 레드/네이비 이색 파티클 상승 (22개)
  - 레드-골드 수평 라인 시머 (3.5s)
  - 코너 장식: 레드/골드 교차 펄스
  - 하단 스크롤 티커 (36s)
  - SIEM/SOAR/SOC/XDR/CSIRT 컬러 배지 팝인
  - SOC 상태 표시등 블링크
- 설정 가능 항목: 위젯 제목, 배너 메인 타이틀, 스크롤 티커 텍스트
- 다국어 지원: 한국어, 영어, 일본어, 중국어(간체)
